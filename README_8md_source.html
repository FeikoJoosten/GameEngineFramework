1 # Game Engine Framework
2 
3 [Docmentation:](https://feikojoosten.github.io/GameEngineFramework/)
4 This project is a free to use C++ engine framework.
5 
6 This framework is a basis that can be used for the creation of a custom engine.
7 This framework was created over the time period of around 16 weeks.

9 This framework DOES NOT include an audio player implementation.

13 # How to use
14 Set the `Game` project as the startup project and make sure the x64 target is selected.
15 A window should pop up with a simple menu stating a `Quit Game` button.
16 
17 By pressing `left ctrl`, `left shift` and `e` you can open a simple developer menu with pretty much no functionality.
18 
19 By opening `Engine/Utility/Defines.hpp` you can change the graphics API from `OpenGL` to `Vulkan` by changing `RENDERER OpenGL` to `RENDERER Vulkan`.
20 To support the Vulkan renderer, be sure to install the [VULKAN SDK](https://vulkan.lunarg.com/).
